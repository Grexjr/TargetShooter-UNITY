<h1>TARGET SHOOTER DEMO</h1>

SUMMARY:
This is a target shooting game in 3d space made with Unity. The player
is a stationary turret, and around the player arise enemies in waves. Shoot
all enemies to move to the next wave and increase your score. 


HOW TO PLAY:
Look around - move mouse
Shoot - left click
Use Powerup - space bar


FEATURES:
This game will feature:

1. Enemies that spawn at random locations and move towards the player
2. An increasing difficulty curve where waves further along get harder
3. A power up to help the player that they need to shoot + extra points
4. An aesthetic theme that the models are based on (sci-fi, garden, etc.)

After V1, this game will also include:

1. A cool skybox and environment, possibly with different levels depending
on the wave
2. Different enemy types that need to be dealt with differently (+ bosses)
3. Permanent upgrade paths
4. Effects, including with powerups, enemy death, and player damage/death
